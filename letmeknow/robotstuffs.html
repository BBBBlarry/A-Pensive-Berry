Homework

  • Make a program that asks or loads all of your family names (just first names).

  • Write a function that takes a vector of the names, plus which number you want to print out (like the 3rd name, for example), then prints out that name.

    • Bonus:

  • Print out how many names you entered.

  • Print out all the names UP TO that number.

  • Print out all the names AFTER that number.

  • MORE BONUS: Sort the names.

  • MORE BONUS: Pass first AND last names (but in ONE vector).
